# CMOS Inverter Design – Schematic to Layout (90nm)

## 📌 Overview
Full VLSI design flow of CMOS Inverter at 90nm technology
node using Cadence Virtuoso, from schematic to physical layout.

---

## ⚙️ What I Did
- Schematic design of CMOS Inverter
- DC Analysis (Voltage Transfer Characteristics)
- Transient Analysis (Switching behavior)
- Layout Design in Cadence Virtuoso
- DRC Check → 0 violations
- LVS Check → Schematic & Layout matched
